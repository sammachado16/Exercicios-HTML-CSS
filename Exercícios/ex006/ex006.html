Principais formatações do HTML5

Negrito / Destaque

Pode ser usada a tag b, que não tem semântica;

Ou pode ser usada a tag strong, que tem semântica.

Itálico / Ênfase

Pode ser usada a tag i, que não tem semântica;

Ou pode ser usada a tag em, que tem semântica.

Texto marcado

Pode ser usada a tag mark, que deixa o texto marcado.

Texto grande e pequeno

Existem tags para criar textos grandes e textos pequenos. Porém, estão obsoletas, especialmente big.

Texto deletado

A tag para um texto deletado é del, que o risca.

Texto inserido

Pode ser usada a tag não semântica u;

Mas também pode ser usada a tag semântica ins

Texto sobrescrito

Tag usada para ocasiões como x2 + 3x - 0, a tag sup.

Texto subscrito

Tag usada para ocasiões como H2O, a tag sub.
